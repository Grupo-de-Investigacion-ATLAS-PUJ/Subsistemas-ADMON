Panel Name

fwTrendingSaveAsSelector.pnl
Introduction

This panel is used to select a data point to save the plot or page configuration to.
Instructions

When you select "Save As..." this dialog will appear, usually with a proposed name for the data point name.

In the text field you can enter any valid data point to which the configuration will be saved. If this is an existing data point you will be warned that it will overwrite the previous configuration. If it is a data point which does not exist, this data point will be created for you.

Before to be able to save the configuration, you should select one available node in the Trend Tree (Clipboard node not allowed).

If you need to create a new Tree node, or see a Trend page/plot configuration before to overwrite it, you could press right click. Two options will be available:

  • View: If is a valid Plot or Page configuration, it will display it
  • Add Node: Will create a node under your selected node

Once you have entered the correct data point name and selected a node, OK button will enable to save your configuration. Or press "Cancel" to close the panel without saving the configuration.
Restrictions None
